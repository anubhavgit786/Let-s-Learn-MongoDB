# **MongoDB Command Line Database Tools Download**


[Download](https://www.mongodb.com/try/download/database-tools)

1. Download Mongo Shell. 
2. Unzip the file.
3. Copy the folder in C drive. 
4. Add the path ```C:\mongodb-database-tools-windows-x86_64-100.7.3\mongodb-database-tools-windows-x86_64-100.7.3\bin``` in environment variables.
